# Emotilt
🎉 내 주변의 이모틸트(Emotilt) 사용자를 찾아 익명으로 이모지와 짧은 메시지를 보내보세요! 😎

### Requirements
- iOS 16.0+ (Xcode 14.0+ required for building project)
- Ultra Wideband (UWB) chip이 내장된 iPhone 11 이후

### Tech Stack
- SwiftUI
- Nearby Interaction
